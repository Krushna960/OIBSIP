# Unemployment Analysis with Python

This project is part of the Oasis Infobyte Data Science Internship.

## Objective
To analyze unemployment trends in India and study the impact of COVID-19 using data visualization.

## Dataset
- Unemployment in India dataset (CMIE-based)

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Insights
- Sharp rise in unemployment during COVID-19
- Urban unemployment is more volatile than rural
- Regional disparities exist across India

## Conclusion
The analysis highlights how economic disruptions affect employment and how data science can help in policy insights.
