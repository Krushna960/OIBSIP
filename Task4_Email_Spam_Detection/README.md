# Email Spam Detection with Machine Learning

This project is part of the Oasis Infobyte Data Science Internship.

## Objective
To classify emails as spam or not spam using machine learning and NLP techniques.

## Dataset
- SMS Spam Collection Dataset

## Algorithm Used
- Multinomial Naive Bayes

## Techniques Used
- Text Vectorization (Bag of Words)

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Results
- Achieved accuracy of ~98%
- Successfully classified spam messages

## Conclusion
The model effectively detects spam messages and demonstrates real-world use of NLP in machine learning.
